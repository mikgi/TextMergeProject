# Quick Start Guide

## 🚀 Fastest Way to Get Started

### 1. Install

```bash
pip install -r requirements.txt
```

### 2. Launch GUI

```bash
python gui_launcher.py
```

### 3. Use GUI

1. Click "Browse Folder" → Select folder with .txt files
2. (Or drag & drop folder into the app)
3. Select export format (Both/DOCX/PDF)
4. Click "▶ Start Merge"
5. Done! Files appear in `output/` folder

---

## Alternative: Command Line

```bash
python main.py ./input_folder output_name --format both
```
